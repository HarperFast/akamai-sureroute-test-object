A component to provide the sureroute-test-object endpoint for Akamai load balancers

For use with Harper **v4.7** or later, make sure to use the appropriate git version tag `1.0.2`:

```
harperdb deploy project=akamai-sureroute-test-object package="git+https://github.com/HarperDB/akamai-sureroute-test-object.git#semver:v1.0.2"
```
